# Marca corrida
Aplicativo para registrar métricas de uma corrida.
Parece um cronometro.

## Metas
- Métricas de corrida
    - [x] tempo
    - [x] passo
    - [ ] velocidade
    - [ ] distancia
- [x] Armazenamento persistente
- Exportar dados
    - [x] apps de terceiros 
- Formato de arquivo
    - [x] JSON
